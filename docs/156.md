在上一课中，我们花了很多时间来真正理解我们的二进制文件内部发生了什么。这为简单的破解奠定了基础。

让我们以写模式启动 radare2。

```
radare2 -w ./0x01_asm_64_helloworld

```

让我们自动分析。

```
aaa

```

求主。

```
s main

```

查看反汇编。

```
v

```

我们看到的内存地址是在磁盘上，因为我们没有运行上一课讨论的二进制文件。

我们看到在 **0xb48** 我们很容易找到我们的字符串。

让我们回到终端视图。

```
q

```

让我们验证字符串。

```
[0x000009e4]> ps @0xb48
Hello World!
[0x000009e4]>

```

让我们黑掉绳子。

```
[0x000009e4]> w Hacked World @0xb48

```

我们来验证一下黑客攻击。

```
[0x000009e4]> ps @0xb48
Hacked World
[0x000009e4]>

```

我们退出 radare2 吧。

```
q

```

现在让我们再次运行我们的二进制！

```
./0x01_asm_64_helloworld
Hacked World

```
